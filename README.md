# Customer Churn Prediction Using Machine Learning

##  Project Overview

Customer churn is one of the major challenges faced by businesses. Losing existing customers can impact revenue and growth. This project uses Machine Learning techniques to predict whether a customer is likely to leave a service based on customer behavior and historical data. The insights generated from this model can help businesses take proactive actions to improve customer retention.

---

##  Problem Statement

Many businesses struggle to identify customers who are likely to discontinue their services. The goal of this project is to build a Machine Learning model that can accurately predict customer churn and help organizations implement effective retention strategies.

##  Dataset Features

The dataset contains customer-related information such as:

- Age
- Gender
- Tenure
- Usage Frequency
- Support Calls
- Payment Delay
- Subscription Type
- Contract Length
- Total Spend

### Target Variable

- 1 → Customer Will Churn
- 0 → Customer Will Stay

##  Project Workflow

### 1. Data Collection
Collected customer churn dataset for analysis and prediction.

### 2. Data Preprocessing
- Handled missing values
- Removed duplicate records
- Encoded categorical features
- Prepared data for model training

### 3. Exploratory Data Analysis (EDA)
Performed visual analysis using:
- Histogram
- Count Plot
- Box Plot
- Correlation Heatmap

### 4. Feature Engineering
Created additional features to improve prediction performance.

### 5. Model Training
Trained and tested multiple Machine Learning models:
- Linear Regression
- K-Nearest Neighbors (KNN)
- Decision Tree
- Random Forest Classifier

### 6. Model Evaluation
Evaluated models using:
- Accuracy Score
- Confusion Matrix
- Classification Report

### 7. Prediction
Predicted whether a customer is likely to churn or stay.

---

##  Machine Learning Model Used

### Random Forest Classifier

Random Forest was selected because:

- Provides high prediction accuracy
- Reduces overfitting
- Handles large datasets efficiently
- Performs well for classification problems
- Produces reliable and stable results

---

##  Results

The Random Forest model successfully predicted customer churn with strong performance and helped identify customers who are at risk of leaving the service.
